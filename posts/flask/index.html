flask備忘

October 06, 2019

Flaskバージョン
Flask==1.0.2

仮想環境にするとき

python3 -m venv flaskworkd
. bin/activate

仮想環境から抜けるとき

deactivate

最小限のコード(main.py)

from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello_world():
return 'hello flask'

if __name__=='__main__':
app.debug = True
app.run()

起動

python3 main.py
