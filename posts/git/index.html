git備忘

September 29, 2019

git checkout id
作業ディレクトリの内容を過去のものに変更できる

git checkout master
最後は，HEADをmasterに戻す
そうしないとdetached HEADと言われる

git add .
git commit –amend
commit後に小さな変更をして，commitを修正したいとき

git commit –amend –no-edit
エディタが開かない

git commit –amend -m “fugafuga”
commitメッセージの修正

git pull
変更ないのにpullしても，メッセージが出るだけなので，常にpullしておけばよい
